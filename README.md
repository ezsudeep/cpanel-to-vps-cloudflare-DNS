# Cloudflare + cPanel Email DNS Guide

This repository provides a simple, easy-to-follow guide for configuring DNS when using Cloudflare for your domain while keeping email hosted on cPanel.  
It covers the correct setup of MX, SPF, DKIM, DMARC, A records, and SRV records.  
Includes clear instructions on when to use **Proxied (orange cloud)** and when to use **DNS Only (gray cloud)** in Cloudflare.  
The guide prevents common issues like email delivery failure, Gmail SPF/DKIM rejection, and broken webmail/cPanel logins.  
Perfect for developers, agencies, hosting providers, and clients migrating to Cloudflare.
